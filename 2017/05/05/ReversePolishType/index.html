Reverse polish type algorithm

逆波兰式算法

说的通俗一点就是把中叙表达式转为右叙表达式。举个例子他可以解析数学表达式。
假设一个数学表达式：56+78*(25+69/3)

他的中叙表达式：[56,+,78,*,(,25,+,69,/,3,)]

他的右叙表达式：[56,78,25,69,3,/,+,*,+]

看到有序表达式之后机器可以从左到右依次执行得到结果
